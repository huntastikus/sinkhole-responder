Trust-store guide

Windows

Download it from the TLS page, save the file as ca.cert.pem, then use either method below.

Lab/home only

Only trust this CA on devices you control in a lab/home network. It lets this tool impersonate any HTTPS site to that device.

Command line

  1. Open Command Prompt as Administrator.
  2. Run certutil -addstore -f Root ca.cert.pem

Certificate Manager

Run certlm.msc → Trusted Root Certification Authorities → Import, then select ca.cert.pem.
