Trust-store guide

macOS

Download it from the TLS page and save the file as ca.cert.pem.

Lab/home only

Only trust this CA on devices you control in a lab/home network. It lets this tool impersonate any HTTPS site to that device.

Keychain Access

Open Keychain Access → System → drag ca.cert.pem in → double-click the certificate → Trust → set “When using this certificate” to “Always Trust”.

Command line

sudo security add-trusted-cert -d -r trustRoot -k /Library/Keychains/System.keychain ca.cert.pem
