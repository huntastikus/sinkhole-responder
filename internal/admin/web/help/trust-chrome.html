Chrome / Chromium

Lab/home only

Only trust this CA on devices you control in a lab/home network. It lets this tool impersonate any HTTPS site to that device.

Windows and macOS

Chrome and Chromium use the OS trust store. Follow the Windows guide or macOS guide.
