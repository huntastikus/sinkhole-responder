Trust-store guide

iOS / iPadOS

Download it from the TLS page, then email or AirDrop ca.cert.pem to the device.

Lab/home only

Only trust this CA on devices you control in a lab/home network. It lets this tool impersonate any HTTPS site to that device.

Both steps are required

  1. Install the profile: Settings → Profile Downloaded.
  2. Then enable full trust: Settings → General → About → Certificate Trust Settings → enable full trust for the certificate.

Installing the profile alone is not enough. The second step is easily missed.
