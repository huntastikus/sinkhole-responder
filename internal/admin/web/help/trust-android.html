Trust-store guide

Android

Download it from the TLS page and transfer ca.cert.pem to the device.

Lab/home only

Only trust this CA on devices you control in a lab/home network. It lets this tool impersonate any HTTPS site to that device.

Install the CA

Open Settings → Security → Encryption & credentials → Install a certificate → CA certificate → pick ca.cert.pem.

Android 7+: apps do not trust user-installed CAs by default. Browsers, such as Firefox with its own trust store, do.
