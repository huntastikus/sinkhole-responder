Trust-store guide

Firefox (own trust store)

Download it from the TLS page and save the file as ca.cert.pem.

Lab/home only

Only trust this CA on devices you control in a lab/home network. It lets this tool impersonate any HTTPS site to that device.

Import the CA

Open Settings → Privacy & Security → Certificates → View Certificates → Authorities → Import → select ca.cert.pem → check “Trust this CA to identify websites”.
